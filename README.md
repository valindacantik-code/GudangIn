# GudangIn
GudangIn adalah aplikasi manajemen gudang sederhana yang dibuat menggunakan bahasa pemrograman Go.

Fitur:
- Tambah Barang
- Ubah Barang
- Hapus Barang
- Transaksi Masuk
- Transaksi Keluar
- Riwayat Transaksi
- Sequential Search
- Binary Search
- Selection Sort
- Insertion Sort
- Statistik Gudang

Dibuat untuk memenuhi Tugas Besar Algoritma dan Pemrograman.
